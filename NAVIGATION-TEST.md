## 🐛 Если что-то не работает:

### **Проблема: "Компания" или "Проект" не открываются**
**Решение:** Убедитесь что Info запущен на порту 3003:
```bash
netstat -ano | findstr ":3003"
```

### **Проблема: "Калькулятор" не показывается**
**Решение:** 
1. Откройте консоль браузера (F12)
2. Проверьте есть ли ошибки
3. Перезагрузите страницу (Ctrl+F5)

### **Проблема: Dashboard не открывается после логина**
**Решение:**
1. Откройте консоль браузера (F12)
2. Проверьте что токены сохранены:
```javascript
console.log('Token:', localStorage.getItem('ipg_token'));
console.log('Refresh Token:', localStorage.getItem('ipg_refresh_token'));
```
3. Если токенов нет - очистите localStorage и войдите заново:
```javascript
localStorage.clear();
window.location.reload();
```
